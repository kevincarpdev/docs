## Tech stack

The Rust-Oleum HOME storefront is built on:

| Layer | Technology | Version |
|-------|-----------|---------|
| **Ecommerce platform** | BigCommerce | Enterprise |
| **Theme framework** | Stencil (Cornerstone) | 6.16.2 |
| **Templating** | Handlebars v4 | — |
| **CSS** | SCSS (Citadel toolkit) | — |
| **Frontend framework** | React + MUI | React 18, MUI 5 |
| **Widget system** | BigCommerce Widget Builder CLI | Custom fork |
| **Build tool** | Webpack | — |
| **Icons** | Phosphor Icons | 2.0.3 |

## Repositories

| Repo | Purpose |

## Docs site domain

The documentation site is hosted on Mintlify at `sparklabsprojects.mintlify.app`. The `.mintlify.app` subdomain slug is set at project creation time and **cannot be changed** through the dashboard.

To get a `rustoleumhome` subdomain, either:

1. **Contact Mintlify support** at [hi@mintlify.com](mailto:hi@mintlify.com) to rename the slug
2. **Create a new project** in the Mintlify dashboard with slug `rustoleumhome` pointing at the same GitHub repo
3. **Set up a custom domain** (e.g., `docs.rustoleumhome.com`) in the Mintlify dashboard under **Settings > Domain setup**, then configure a DNS CNAME record
