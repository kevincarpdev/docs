## What this repo is

This repository is the Rust-Oleum HOME storefront monorepo. It contains:

- the shared API, UI, widget, and tooling packages
- the `rustoleum-home` BigCommerce Stencil theme
- the Mintlify docs site for developers and content teams

It does **not** use the older split workflow of a separate widget-builder repo plus a separate theme repo.

<Warning>
  Older notes that mention `npm`, a separate `widget-builder` repository, `meta.json`, sync scripts, or
  `stencil init` are out of date for this repo.
</Warning>

## Current stack

| Area | Current surface |
| --- | --- |
| Package manager | `bun` |
| Language | TypeScript |
| Commerce platform | BigCommerce |
| Theme runtime | Stencil theme under `apps/storefronts/rustoleum-home/stencil-theme` |
| Shared UI | React 19 + MUI 7 in `packages/ui` |
| Shared widgets | `packages/widgets` |
| Publish and placement tooling | `packages/tooling` |
| Documentation | Mintlify under `mintlify-docs` |

## Monorepo layout

| Path | Responsibility |
| --- | --- |
| `packages/api` | Shared API utilities and API-facing code |
| `packages/ui` | Shared React primitives, composites, icons, and theme tokens |
| `packages/widgets` | Package-backed Page Builder widgets, manifests, templates, and widget-local components |
| `packages/tooling` | Widget and placement CLI, publish flow, init flow, recovery commands |
| `apps/storefronts/rustoleum-home` | Storefront-local env files and the active Stencil theme |
| `apps/storefronts/rustoleum-home/stencil-theme` | Theme runtime, registry, wrappers, Handlebars templates, bundle and push scripts |
| `mintlify-docs/dev` | Developer documentation |
| `mintlify-docs/content` | End-user and content-team documentation |

## Common commands

```bash
# install workspace dependencies
bun install

# run workspace package watchers + stencil theme
bun run dev

# full workspace build
bun run build

# primary widget publish flow
bun run widgets:publish

# push the stencil theme bundle
bun run stencil:push
```
