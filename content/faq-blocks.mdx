## FAQ widget types

There are several FAQ widgets available, each designed for a different context:

| Widget | Best used for |
|--------|--------------|
| **FAQHeaderBlock** | Page header with title and introductory text for FAQ sections |
| **FaqBlock** | Simple accordion-style Q&A section (general purpose) |
| **FaqHighlightBlock** | Featured FAQ with visual emphasis — used for highlighting key questions |
| **FAQPageContentBlock** | Full-page FAQ content with structured categories |
| **FAQPageContent** | Alternative full-page FAQ layout |

## FaqBlock (general purpose)

The most common FAQ widget. Displays questions and answers in an expandable accordion format.

### Settings

| Field | Description |
|-------|-------------|
| **Section Title** | Heading above the FAQ section |
| **Question N** | The question text |
| **Answer N** | The answer text (supports basic formatting) |

### Usage

- Product pages — Common questions about the product
- Category pages — General questions about a surface type
- Info pages — Shipping, returns, warranty FAQs

## FaqHighlightBlock

A visually prominent FAQ section that draws attention to the most important questions.

### When to use

- Homepage or landing pages where you want to feature 2-3 key questions
- Pages where the FAQ needs to stand out from surrounding content

## FAQHeaderBlock

A header-only widget that provides a title and optional description above FAQ content.

### Settings

| Field | Description |
|-------|-------------|
| **Heading** | Main title (e.g., "Frequently Asked Questions") |
| **Subheading** | Optional description text |

### Usage

Place above a FaqBlock or FAQPageContentBlock to provide context.

## FAQPageContentBlock

A comprehensive FAQ layout designed for dedicated FAQ pages. Supports categorized questions with a structured layout.

### Settings

Organized by category sections, each containing multiple Q&A pairs.

### Usage

- Dedicated FAQ pages (e.g., `/faq`)
- Knowledge base style pages with many questions organized by topic

## Best practices

- Keep answers concise — link to detailed pages for complex topics
- Use consistent formatting across all FAQ sections
- Order questions from most common to least common
- Use the FAQHeaderBlock at the top of any page with multiple FAQ sections
