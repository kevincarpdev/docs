### Image Guidelines

## Title Section

A flexible page title and introductory copy section with optional subtitle, coupon/discount label, and CTA. Used at the top of category pages, campaign pages, and any page that needs a structured header below the hero.

### Fields

| Field | Label in UI | Description |
|-------|-------------|-------------|
| `heading` | Heading | Primary page or section title â€” supports rich text |
| `body` | Body | Supporting copy beneath the heading â€” supports rich text |
| `subtitle` | Subtitle | Optional small label above the heading |
| `variant` | Variant | Controls the visual style/layout of the title section |
| `ctaType` | Type | Select the CTA type: **Button** or **Coupon** |
| `anchor` | Anchor (Settings tab) | Optional `#id` for deep linking |

**When CTA Type = Button:**

| Field | Label in UI | Description |
|-------|-------------|-------------|
| `buttonText` | Button Text | CTA button label |
| `url` | URL | CTA destination |

**When CTA Type = Coupon:**

| Field | Label in UI | Description |
|-------|-------------|-------------|
| `couponCopy` | Coupon Copy | The coupon or promo code to display (e.g., `SAVE20`) |
| `couponCopyActionLabel` | Coupon Button Label | Label for the copy-to-clipboard button (e.g., "Copy code") |
| `couponCopiedActionLabel` | Coupon Copied Label | Confirmation label after copying (e.g., "Copied\!") |

### Guidelines

- Use **Subtitle** to add a small category label or campaign tag above the main heading (e.g., "Spring Collection").
- **Coupon** type is for promotional pages where a discount code needs to be prominently displayed and easily copied.
- For most pages, `ctaType` should be **Button** â€” the coupon option is specifically for promo campaigns.
- Keep the heading under 70 characters for clean rendering at all breakpoints.
