## Overview

The Hero Carousel (also called Hero Section Carousel) is the large banner at the top of the homepage. It supports multiple slides, each with its own content, images, and call-to-action buttons.

## Editing a slide

Click the carousel on the homepage in Page Builder to open its settings. Each slide has its own tab (e.g., "Slide 1 Content").

### Slide fields

| Field | Description | Example |
|-------|-------------|---------|
| **Badge** (optional) | Small label above the headline | `NEW` |
| **Badge Subtext** (optional) | Text below the badge | `Just Launched` |
| **Badge Product Link** (optional) | Product to link the badge to | Search and select a product |
| **Headline** | Main heading text | `Transform Any Room with Rust-Oleum HOME` |
| **Subheadline** | Supporting description text | `Refresh your cabinets, countertops, and floors...` |
| **Background Image** | Full-width background image for the slide | Upload via Image Manager |
| **Panel Background Color** | Color of the text panel overlay | `#5F6D68` |
| **Primary Button Text** | Label for the main CTA button | `Explore All Products` |
| **Primary Button URL** | Link destination for the primary button | `/shop/all` |
| **Secondary Button Text** (optional) | Label for a second button | `Room Visualizer` |
| **Secondary Button URL** (optional) | Link destination for the secondary button | `/room-visualizer` |
| **Footer Text** (optional) | Small text below the buttons | `See how Rust-Oleum HOME looks in your space` |

### Background image guidelines

- **Recommended size**: At least 1920px wide for full-width display
- **Format**: JPEG for best compression on photographs
- **Aspect ratio**: Approximately 16:9 or wider
- The text panel overlays the left side of the image, so place the visual focal point on the right side

### Adding more slides

Each carousel supports multiple slides. Look for additional "Slide N Content" tabs in the settings panel. Configure each slide independently.

## Carousel settings

At the top level of the widget settings, you may find carousel-wide controls:

| Setting | Description |
|---------|-------------|
| **Show Arrows** | Display left/right navigation arrows |
| **Autoplay** | Automatically cycle through slides |
| **Interval** | Time between auto-advance (in milliseconds) |
| **Show Dots** | Display navigation dots below the carousel |
