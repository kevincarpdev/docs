title: "Widget Settings Reference"
description: "Every field available in the Page Builder settings panel (⋯ menu) for each widget."
---

When you click **⋯ → Settings** on a widget in Page Builder, a panel opens on the right. This page documents every field in that panel — what it does, what type it is, and when it appears (some fields are conditional and only show when another field is set a certain way).

<Note>
  Fields marked **(optional)** can be left blank without breaking the widget. Fields marked **(conditional)** only appear when another field is set to a specific value.
</Note>

---

## Hero Carousel

**Array:** Slides (repeatable — add as many slides as needed)

Each slide has these fields:

| Field | Type | Notes |
|---|---|---|
| Badge | Text | Short label above the headline (optional) |
| Badge Subtext | Text | Smaller text below the badge (optional) |
| Badge URL | Text | Makes the badge a clickable link (optional) |
| Headline | Text | Main slide heading |
| Subheadline | Text | Supporting line below the headline |
| Background Image | Image | Full-bleed slide background |
| Panel Background Color | Text | Hex color for the content panel (default: `#5F6D68`) |
| Primary Button Text | Text | CTA button label |
| Primary Button URL | Text | CTA button destination |
| Secondary Button Text | Text | Second CTA label (optional) |
| Secondary Button URL | Text | Second CTA destination (optional) |
| Footer Text | Text | Small text below the CTAs (optional) |
| Footer Link | Text | Makes footer text a link (optional) |

**Settings tab:**

| Field | Type | Notes |
|---|---|---|
| Anchor | Text | Sets a `#anchor` ID on this widget for in-page links (optional) |

## Hero

**Layout variants:** `responsive`, `full-width`, `homepage`, `roomvo`

**Array:** Media Items (image or video per item)

Each media item has:

| Field | Type | Notes |
|---|---|---|
| Type | Select | `Image` or `Video` |
| Image | Image | (conditional: shown when Type = Image) |
| Video URL | Text | (conditional: shown when Type = Video) |
| Homepage Override fields | — | Heading, Subheading, Body, Background Color, Primary CTA, Secondary CTA — these override the global hero copy for this specific slide (homepage variant only) |

**Settings tab sections:**

| Section | Field | Type | Notes |
|---|---|---|---|
| Layout Settings | Layout Variant | Select | `full-width`, `responsive`, `homepage`, `roomvo` |
| Layout Settings | Desktop Arrangement | Select | `Stacked` or `Side by side` — (conditional: responsive only) |
| Layout Settings | Mobile Arrangement | Select | `Stacked image top` or `Stacked image bottom` — (conditional: responsive only) |
| Layout Settings | Homepage Background | Select | Rocky River, Bijou Blue, Marine Magic, Other — (conditional: homepage only) |
| Layout Settings | Custom Background | Color picker | (conditional: Background = Other) |
| Layout Settings | Anchor | Text | Optional in-page anchor ID |
| Label | Label Type | Select | Discount label style — (conditional: responsive or full-width only) |
| Label | Label Text | Text | optional |
| Label | Date | Text | UTC date format (optional) |
| Label | Pill | Text | Pill text — (conditional: homepage only) |
| Copy | Heading | Text | Main hero headline |
| Copy | Subheading | Text | Supporting line (optional) |
| Copy | Body | Text | Body copy |
| Surface Selector | Show surface selector | Toggle | Enables Floor/Cabinet/Countertop tabs — (conditional: responsive only) |
| Surface Selector | Tab 1–3 Label | Text | Tab label (conditional: surface selector on) |
| Surface Selector | Tab 1–3 Link | Text | Tab destination URL (conditional: surface selector on) |
| Surface Selector | Tab 1–3 Icon | Select | Floor, Cabinet, Countertop, Other, None (conditional: surface selector on) |
| Surface Selector | Tab 1–3 Active | Toggle | Pre-selects this tab (conditional: surface selector on) |
| CTA | Destination Type | Select | Page URL or Product — (conditional: responsive, full-width, homepage) |
| CTA | Text | Text | Button label (optional) |
| CTA | Left/Right Icon | Select | Phosphor icon name |
| CTA | Variant | Select | Button style |
| Secondary CTA | (same CTA fields) | — | (conditional: homepage only) |
| Secondary CTA | Tagline | Text | Small text below the secondary CTA (optional) |
| Roomvo CTA | (CTA fields without destination) | — | (conditional: roomvo only) |
