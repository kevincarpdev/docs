When you click **⋯ → Settings** on a widget in Page Builder, a panel opens on the right. This page documents every field in that panel — what it does, what type it is, and when it appears.

<Note>
  Fields marked **(optional)** can be left blank. Fields marked **(conditional)** only appear when another field is set to a specific value — if you don't see a field, check whether the condition it depends on is met.
</Note>

## Hero Carousel

A full-width homepage banner that cycles through multiple slides automatically or on user interaction. Each slide has its own image, copy, and CTA buttons.

**Array:** Slides (repeatable — add as many slides as needed)

Each slide has:

| Field | Type | Notes |
|---|---|---|
| Badge | Text | Short label above the headline (optional) |
| Badge Subtext | Text | Smaller text below the badge (optional) |
| Badge URL | Text | Makes the badge a clickable link (optional) |
| Headline | Text | Main slide heading |
| Subheadline | Text | Supporting line below the headline |
| Background Image | Image | Full-bleed slide background |
| Panel Background Color | Text | Hex color for the content panel (default: `#5F6D68` Rocky River) |
| Primary Button Text | Text | CTA button label |
| Primary Button URL | Text | CTA button destination |
| Secondary Button Text | Text | Second CTA label (optional) |
| Secondary Button URL | Text | Second CTA destination (optional) |
| Footer Text | Text | Small text below the CTAs (optional) |
| Footer Link | Text | Makes footer text a clickable link (optional) |

**Settings tab:**

| Field | Type | Notes |
|---|---|---|
| Anchor | Text | Sets a `#anchor` ID on this widget for in-page links (optional) |

## Hero

A versatile hero section used across landing pages and the homepage. Choose a layout variant first — it controls which other fields appear.

**Layout variants:**

| Variant | What it looks like |
|---|---|
| **responsive** | Standard landing page hero. Image and copy sit side-by-side on desktop (configurable), stacked on mobile. Supports a surface selector (Floor / Cabinet / Countertop tabs) to link shoppers to different product surfaces. |
| **full-width** | Edge-to-edge image with copy overlaid on top. Used for bold, cinematic hero moments. |
| **homepage** | The homepage-specific hero. Includes a branded color background panel, pill/label treatment, per-slide copy overrides, and a secondary CTA with tagline. |
| **roomvo** | A centered layout that triggers the Roomvo room visualizer tool. The CTA opens the Roomvo experience instead of navigating to a page or product. |

**Array:** Media Items (image or video per item — multiple items become a carousel)

Each media item:

| Field | Type | Notes |
|---|---|---|
| Type | Select | `Image` or `Video` |
| Image | Image | (conditional: shown when Type = Image) |
| Video URL | Text | (conditional: shown when Type = Video) |
| Homepage Override: Heading | Text | Per-slide heading (overrides global heading for this slide — homepage only) |
| Homepage Override: Subheading | Text | Per-slide subheading (homepage only) |
| Homepage Override: Body | Text | Per-slide body (homepage only) |
| Homepage Override: Background Color | Select | Rocky River, Bijou Blue, Marine Magic, Other, or use global setting (homepage only) |
| Homepage Override: Primary CTA | — | Full CTA fields per slide (homepage only) |
| Homepage Override: Secondary CTA | — | Full secondary CTA fields per slide (homepage only) |

**Settings tab:**

| Section | Field | Type | Conditional |
|---|---|---|---|
| Layout | Layout Variant | Select | — |
| Layout | Desktop Arrangement | Select | Stacked or Side by side — responsive only |
| Layout | Mobile Arrangement | Select | Image top or Image bottom — responsive only |
| Layout | Homepage Background | Select | Rocky River, Bijou Blue, Marine Magic, Other — homepage only |
| Layout | Custom Background | Color picker | shown when Background = Other |
| Layout | Anchor | Text | — |
| Label | Label Type | Select | Discount label style — responsive and full-width only |
| Label | Label Text | Text | — |
| Label | Date (UTC) | Text | Countdown date — responsive and full-width only |
| Label | Pill Text | Text | homepage only |
| Copy | Heading | Text | Global heading for all slides |
| Copy | Subheading | Text | (optional) |
| Copy | Body | Text | — |
| Surface Selector | Show surface selector | Toggle | responsive only |
| Surface Selector | Tab 1–3 Label | Text | shown when surface selector is on |
| Surface Selector | Tab 1–3 Link | Text | shown when surface selector is on |
| Surface Selector | Tab 1–3 Icon | Select | Floor, Cabinet, Countertop, Other, None |
| Surface Selector | Tab 1–3 Active | Toggle | Pre-selects this tab by default |
| CTA | Destination Type | Select | Page URL or Product — responsive, full-width, homepage |
| CTA | Text | Text | Button label (optional) |
| CTA | Left / Right Icon | Select | Phosphor icon name |
| CTA | Variant | Select | Button style |
| Secondary CTA | (same fields as CTA) | — | homepage only |
| Secondary CTA | Tagline | Text | Small handwritten-style text below the button — homepage only |
| Roomvo CTA | (CTA fields, no destination) | — | roomvo only |

## How It Works

A step-by-step instructional section. Two very different visual approaches — choose based on content type.

**Layout variants:**

| Variant | What it looks like |
|---|---|
| **categories** | Tabbed interface — up to 3 product categories (e.g. Floor, Countertop, Cabinet), each with up to 4 steps. Shoppers click a tab to see that category's steps. Best for products with multiple application surfaces. |
| **flat** | A linear sequence of steps shown as cards, no tabs. Two sub-styles: **background** (image-led cards with a colored background and CTA) and **transparent** (numbered benefit cards without images or CTAs, clean on white). Best for a single application flow. |

**Settings:**

| Section | Field | Type | Conditional |
|---|---|---|---|
| Layout | Variant | Select | `categories` or `flat` |
| Category 1–3 Name | Text | Tab label (e.g. "Floor", "Countertop") | categories only |
| Category 1–3 Steps 1–4: Image | Image | Step image | categories only |
| Category 1–3 Steps 1–4: Title | Text | Step title | categories only |
| Category 1–3 Steps 1–4: Description | Text | Step description | categories only |
| Flat Style | Style | Select | `background` or `transparent` | flat only |
| Flat Style | Body Copy | Text | Intro paragraph | flat only |
| Flat Steps 1–4: Image | Image | — | flat + background only |
| Flat Steps 1–4: Label | Text | Pill label (defaults to "Step 1", "Step 2" etc.) | flat only |
| Flat Steps 1–4: Title | Text | — | flat only |
| Flat Steps 1–4: Copy | Text | — | flat + background only |
| Flat Steps 1–4: Icon | Select | Phosphor icon | flat only |
| CTA | (compose.button fields) | — | flat + background only |

## Step Section

A sophisticated multi-step section with sticky side navigation. Shoppers click step names in the sidebar to jump to each step, which expands to show sub-steps (image + copy). Used for detailed application instructions on landing pages.

**Array:** Items (repeatable — mix of step headers, sub-steps, and learn-more links)

Each item has a Type field:
- **step** — a top-level step (appears in the sticky navigation sidebar)
- **substep** — content card that belongs to a step (image + copy)
- **learnMore** — a link card shown below the steps

| Field | Type | Conditional |
|---|---|---|
| Item Type | Select | step / substep / learnMore |
| Step ID | Text | step only — used to link substeps to this step |
| Heading | Text | step only |
| Navigation Name | Text | step only — label shown in sidebar nav |
| Parent Step ID | Text | substep only — must match a Step ID above |
| Image | Image | substep only |
| Sub-Step Heading | Text | substep only |
| Sub-Step Body | Text | substep only (supports markdown) |
| Icon | Select | learnMore only |
| Label | Text | learnMore only |
| Link | Text | learnMore only |

**Settings tab:**

| Field | Type | Notes |
|---|---|---|
| Anchor | Text | Optional in-page anchor ID |
| Section Aria Label | Text | Accessibility label for the section |
| Learn More Label | Text | Heading shown above the learn-more links |

## Product Grid

A grid of product cards pulled from the BigCommerce catalog. Two display modes — choose based on how many products you're featuring and how much visual weight you want.

**Layout variants:**

| Variant | What it looks like |
|---|---|
| **normal** | Standard product grid with optional filter chips, section heading, and pagination. Works for 4+ products. Configurable rows (1 row = 4 products on desktop). Mobile can scroll horizontally or stack vertically. |
| **rich** | A featured product display for 1–3 specific products. Each product can have custom CTA text and up to 6 gallery images. Better for highlighted or featured products on landing pages. |

**Array:** Products (repeatable)

Each product entry (used in both variants):
- Product ID
- Filter CSV (normal variant — comma-separated filter category names)
- CTA Text (rich variant — custom button label)
- Rich Images 1–6 (rich variant — gallery images for this product)

**Settings:**

| Section | Field | Type | Notes |
|---|---|---|---|
| Copy | Heading | Text | Section heading (normal variant) |
| Copy | Subheading | Text | Supporting line (normal variant) |
| Layout | Layout Variant | Select | `normal` or `rich` |
| Filters | Enable Filters | Toggle | Shows filter chip row (normal only) |
| Pagination | Pagination Count | Range | Products per page (normal only) |
| Settings | Anchor | Text | Optional in-page anchor ID |
