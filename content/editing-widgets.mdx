## Opening widget settings

1. Make sure you are in **Design** mode (not Preview)
2. Click on any widget on the page canvas
3. The **settings panel** appears on the left sidebar

The settings panel shows the widget name at the top, followed by tabs and sections of editable fields.

## Common field types

### Text input

Single-line text fields for headings, button labels, URLs, etc. Type directly into the field.

### Text area

Multi-line text for descriptions and body copy. Supports longer content.

### Markdown formatting

Some text fields support basic markdown syntax for bold, italic, lists, links, and more. See the full [Markdown Formatting guide](/content/markdown-formatting) for syntax, examples, and a list of which widget fields support it.

### Image Manager

Click the image area to open BigCommerce's Image Manager where you can:
- Upload a new image
- Select from previously uploaded images
- Enter an image URL

Recommended image formats: JPEG for photos, PNG for graphics with transparency, WebP for optimized delivery.

### Product search

Type a product name to search and select a product. The widget will pull product data (name, image, price, URL) automatically.

### Color picker

Click the color swatch to open a color picker. Enter a hex code (e.g., `#5F6D68`) or use the visual picker.

### Dropdown / Select

Choose from a list of predefined options (e.g., "Yes" / "No" for toggle fields).

### Toggle

On/off switches for optional features like "Show Arrows", "Has Article Link", etc.

## Widget structure

Most widgets organize their settings into:

- **Tabs** — Major sections (e.g., "Slide 1 Content", "Slide 2 Content")
- **Sections** — Groups within a tab (e.g., "Section 1 Settings", "Section 1 - Image 1")
- **Fields** — Individual settings within a section

Navigate between tabs and sections using the headers in the left panel.

## Saving your edits

After making changes:

1. Click **Save** in the top right to save as a draft
2. Click **Publish** to make the changes live

<Note>
  Changes are applied to the widget instance on that specific page. If the same widget type is used on multiple pages, each instance has its own settings.
</Note>

## Reverting changes

Click **Discard** to undo all changes since your last save. This cannot be undone — your changes will be lost.
