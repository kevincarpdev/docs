Landing pages on Rust-Oleum HOME are standard BigCommerce web pages with a **custom page template** assigned. Each template replaces the default two-column layout with a full-width canvas made up of named drop zones — **regions** — where you place widgets in Page Builder.

There are **five landing page types** used across the site. Two of them (Influencer and Trend) share the same underlying template.

<Note>
Landing pages use the same Page Builder widgets documented in the Widget Reference. The template just controls which regions exist and in what order.
</Note>

---

## How to create a landing page

1. Go to **Storefront → Web Pages** in BigCommerce admin
2. Click **Create a Page**
3. Fill in **Page Name** and **Page URL**
4. Scroll to **Template File** and select the appropriate template (see table below)
5. Click **Save**
6. Go to **Storefront → Page Builder**, select the new page from the top-left picker
7. Drag widgets into each region on the canvas
8. Click **Save**, then **Publish**

<Note>
The **Template File** selector is in the Advanced section of the web page form — scroll down if you don't see it immediately.
</Note>

## Page types at a glance

| Page type | Template file | Use for |
|-----------|---------------|---------|
| [Promotional](#promotional) | `landing-campaign` | Sales, seasonal campaigns, product launches |
| [Educational](#educational) | `landing-full-width` | How-to, brand storytelling, long-form editorial |

## Promotional

**Template file:** `landing-campaign`

Use for time-limited promotions, product launches, and seasonal campaigns. Built around a single conversion goal — hero → offer → products → social proof → CTA.

### Regions

| Region | Recommended widgets | Purpose |
|--------|---------------------|---------|
| `landing_campaign_hero` | Hero Carousel, Hero | Primary campaign visual and headline |
| `landing_campaign_offer` | Offer Banner, Horizontal Banner | Promotional details, pricing, or discount callout |
| `landing_campaign_featured_products` | Product Grid, Product Highlights | Products featured in the campaign |
| `landing_campaign_social_proof` | Reviews, UGC, User Post | Customer proof and trust signals |
| `landing_campaign_cta` | Title Section, Offer Banner | Final call to action or coupon |

### Example layout

```
┌─────────────────────────────────────┐
│  Hero Carousel                      │  ← landing_campaign_hero
├─────────────────────────────────────┤
│  Offer Banner (promo/discount)      │  ← landing_campaign_offer
├─────────────────────────────────────┤
│  Product Grid (featured products)   │  ← landing_campaign_featured_products
├─────────────────────────────────────┤
│  Reviews or UGC                     │  ← landing_campaign_social_proof
├─────────────────────────────────────┤
│  Title Section with coupon          │  ← landing_campaign_cta
└─────────────────────────────────────┘
```

## Educational

**Template file:** `landing-full-width`

Use for how-to content, brand storytelling, and long-form editorial. Three free-form content sections let you mix and match widgets freely — it's the most flexible template.

### Regions

| Region | Recommended widgets | Purpose |
|--------|---------------------|---------|
| `landing_full_width_hero` | Hero, Hero Carousel | Page header |
| `landing_full_width_content_section_1` | Image Text, Benefits, Values Block | First editorial block |
| `landing_full_width_content_section_2` | Environment Block, Quote Image, How It Works | Second editorial block |
| `landing_full_width_content_section_3` | Step Section, Comparison, Image Text | Third editorial block |
| `landing_full_width_featured_products` | Product Grid, Product Highlights, Bundle Offer | Product showcase |
| `landing_full_width_gallery` | Gallery, Before & After | Visual proof or inspiration |
| `landing_full_width_testimonials` | Reviews, UGC, User Post | Social proof |
| `landing_full_width_cta` | Title Section, Newsletter, Offer Banner | Page-level CTA |

### Example layout

```
┌─────────────────────────────────────┐
│  Hero (editorial header)            │  ← landing_full_width_hero
├─────────────────────────────────────┤
│  Image Text (brand story)           │  ← landing_full_width_content_section_1
├─────────────────────────────────────┤
│  Benefits (key claims)              │  ← landing_full_width_content_section_2
├─────────────────────────────────────┤
│  Step Section (how to apply)        │  ← landing_full_width_content_section_3
├─────────────────────────────────────┤
│  Product Grid (shop the line)       │  ← landing_full_width_featured_products
├─────────────────────────────────────┤
│  Gallery – Inspiration variant      │  ← landing_full_width_gallery
├─────────────────────────────────────┤
│  UGC                                │  ← landing_full_width_testimonials
├─────────────────────────────────────┤
│  Newsletter                         │  ← landing_full_width_cta
└─────────────────────────────────────┘
```

## Influencer & Trend

**Template file:** `landing-category` (shared by both page types)

Both Influencer and Trend pages use the same template. The difference is in content and positioning — Influencer pages are built around a creator partnership, Trend pages are built around a style or aesthetic moment. The structure is the same: introduce → showcase → products → related content → social proof.

### Regions

| Region | Recommended widgets | Purpose |
|--------|---------------------|---------|
| `landing_category_hero` | Hero, Horizontal Banner | Category or creator header |
| `landing_category_grid` | Categories | Surface or style navigation cards |
| `landing_category_features` | Benefits, Image Text | Key features or editorial callouts |
| `landing_category_products` | Product Grid, Product Highlights | Recommended products |
| `landing_category_related_categories` | Categories | Related styles, trends, or categories |
| `landing_category_testimonials` | Reviews, UGC | Social proof |

### Example layout (Influencer)

```
┌─────────────────────────────────────┐
│  Hero (influencer / collab header)  │  ← landing_category_hero
├─────────────────────────────────────┤
│  Categories (style choices)         │  ← landing_category_grid
├─────────────────────────────────────┤
│  Image Text (editorial story)       │  ← landing_category_features
├─────────────────────────────────────┤
│  Product Grid (shop the look)       │  ← landing_category_products
├─────────────────────────────────────┤
│  Categories (related looks)         │  ← landing_category_related_categories
├─────────────────────────────────────┤
│  UGC or Reviews                     │  ← landing_category_testimonials
└─────────────────────────────────────┘
```

### Example layout (Trend)

```
┌─────────────────────────────────────┐
│  Hero (trend header image)          │  ← landing_category_hero
├─────────────────────────────────────┤
│  Categories (room or surface types) │  ← landing_category_grid
├─────────────────────────────────────┤
│  Benefits (why this trend)          │  ← landing_category_features
├─────────────────────────────────────┤
│  Product Highlights (trend SKUs)    │  ← landing_category_products
├─────────────────────────────────────┤
│  Categories (related trends)        │  ← landing_category_related_categories
├─────────────────────────────────────┤
│  User Post (community content)      │  ← landing_category_testimonials
└─────────────────────────────────────┘
```

## Post-Purchase

**Template file:** `landing-product-story`

Use for post-purchase follow-up pages and single-product deep-dives. Structured to reinforce the purchase: lead with the product, show benefits, demonstrate how to use it, and build confidence with reviews.

### Regions

| Region | Recommended widgets | Purpose |
|--------|---------------------|---------|
| `landing_product_story_hero` | Hero | Product hero — image or video |
| `landing_product_story_benefits` | Benefits, Image Text | Key product benefits and claims |
| `landing_product_story_gallery` | Gallery, Before & After | Before/after results or inspiration |
| `landing_product_story_how_it_works` | How It Works, Step Section | Application steps |
| `landing_product_story_products` | Product Grid, Bundle Offer, Product Application | Related products or complete-the-kit |
| `landing_product_story_testimonials` | Reviews, UGC | Product-specific reviews |
| `landing_product_story_cta` | Title Section, Offer Banner | Next step CTA |

### Example layout

```
┌─────────────────────────────────────┐
│  Hero – video (product video)       │  ← landing_product_story_hero
├─────────────────────────────────────┤
│  Benefits (product claims)          │  ← landing_product_story_benefits
├─────────────────────────────────────┤
│  Before & After                     │  ← landing_product_story_gallery
├─────────────────────────────────────┤
│  Step Section (how to apply)        │  ← landing_product_story_how_it_works
├─────────────────────────────────────┤
│  Bundle Offer (complete the kit)    │  ← landing_product_story_products
├─────────────────────────────────────┤
│  Reviews                            │  ← landing_product_story_testimonials
├─────────────────────────────────────┤
│  Offer Banner (final CTA)           │  ← landing_product_story_cta
└─────────────────────────────────────┘
```

## Tips for all landing pages

- **Always fill the hero region first.** Every template starts with a hero — a page without one looks broken.
- **Regions are fixed top-to-bottom.** You cannot reorder regions within a template. If you need a different order, use a different template or raise it with the dev team.
- **Stack widgets within a region sparingly.** You can place multiple widgets in one region, but keep it to 1–2 to avoid visual clutter.
- **The two-column sidebar is hidden** on all landing page templates. Don't try to use Column 1 or Column 2 regions — they're hidden via CSS on these templates.
- **Anchor links work across all regions.** Set an anchor in any widget's Settings panel (`...` ellipsis) and you can link to it with `#anchor-name` from nav or other widgets.
- **Influencer and Trend use the same template** — their distinction is entirely in the content you put in, not in any structural difference. Use the hero and editorial sections to set the right tone for each.
